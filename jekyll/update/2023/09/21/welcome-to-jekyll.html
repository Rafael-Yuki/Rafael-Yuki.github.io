Os Desafios Escolhidos

Sep 21, 2023

Desafios Concluídos:

  1. Desafio de Configuração Inicial do GitHub: Configure um arquivo de configuração .gitignore para excluir arquivos desnecessários do repositório.

  2. Desafio de Branch Protegida: Configure a branch principal como protegida, exigindo revisões de código antes de mesclar pull requests.
  3. Desafio de Fluxo de Trabalho Customizado: Crie um fluxo de trabalho personalizado no GitHub Actions para um cenário específico do seu projeto.
  4. Desafio de Integração com API: Integre o GitHub Actions com uma API externa, como um serviço de notificação.
  5. Desafio de Integração Contínua Multi-Plataforma: Configure o pipeline de CI para executar testes em diferentes sistemas operacionais (Linux, Windows, macOS).
  6. Desafio de Publicação de GitHub Pages: Configure o pipeline para publicar automaticamente o site estático no GitHub Pages.
  7. Desafio de Análise de Desempenho: Utilize ferramentas como Lighthouse para avaliar o desempenho do site e implemente otimizações.
